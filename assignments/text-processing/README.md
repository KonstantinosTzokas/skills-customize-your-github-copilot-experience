# 📘 Assignment: Python Text Processing

## 🎯 Objective

Master Python's string manipulation capabilities, file I/O operations, and text processing techniques. You will build practical programs to handle real-world text processing tasks including reading files, transforming data, and analyzing text patterns.

## 📝 Tasks

### 🛠️	String Manipulation and Text Transformation

#### Description
Write Python functions to perform various string operations including case conversion, substring search, pattern replacement, and string formatting. Practice using built-in string methods and understand how to work with text data efficiently.

#### Requirements
Completed program should:

- Implement functions to convert text to uppercase, lowercase, and title case
- Create a function to search for and count word occurrences in text
- Implement find-and-replace functionality with pattern matching
- Use string methods like `split()`, `join()`, `strip()`, and `replace()`
- Handle edge cases like empty strings and special characters


### 🛠️	File I/O Operations

#### Description
Read from and write to files using Python's file handling methods. Implement programs that process text files, extract information, and save results to new files.

#### Requirements
Completed program should:

- Open and read text files in different modes (read, write, append)
- Process file contents line by line
- Write processed data to output files
- Use proper file handling with context managers (`with` statement)
- Handle file not found and permission errors appropriately


### 🛠️	Text Analysis and Data Processing

#### Description
Analyze text data to extract insights such as word frequency, sentence statistics, and pattern detection. Create programs that summarize, filter, and transform text based on specific criteria.

#### Requirements
Completed program should:

- Count word frequencies and identify the most common words
- Calculate statistics like average word length and sentence count
- Filter text based on criteria (e.g., lines containing specific words)
- Remove duplicates and normalize text formatting
- Generate summaries or reports from text data
