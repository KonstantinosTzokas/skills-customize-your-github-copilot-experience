# 📘 Assignment: Statistics with Python

## 🎯 Objective

Master statistical analysis and data manipulation using pandas and numpy. You will learn to load datasets, perform descriptive statistics, calculate probability distributions, and conduct hypothesis testing to draw meaningful insights from data.

## 📝 Tasks

### 🛠️	Data Loading and Descriptive Statistics

#### Description
Learn to load data from various sources using pandas and compute fundamental statistical measures. Perform exploratory data analysis to understand data distributions, identify patterns, and detect outliers.

#### Requirements
Completed program should:

- Load data from CSV, Excel, and JSON files using pandas
- Access and manipulate dataframe columns and rows
- Calculate descriptive statistics (mean, median, mode, standard deviation, variance)
- Generate summary statistics for entire datasets and grouped data
- Identify and handle missing values appropriately
- Detect outliers using statistical methods
- Create contingency tables and cross-tabulations
- Generate correlation matrices to find relationships between variables


### 🛠️	Data Manipulation and Aggregation

#### Description
Transform and reshape data using pandas operations including filtering, sorting, merging, and grouping. Learn to prepare clean, aggregated datasets for further analysis and visualization.

#### Requirements
Completed program should:

- Filter dataframes based on conditions and criteria
- Sort data by one or multiple columns
- Create new calculated columns from existing data
- Merge and join dataframes from different sources
- Group data and apply aggregation functions (sum, mean, count)
- Pivot data to reshape structure for analysis
- Handle categorical data and encoding
- Perform time-series operations and resampling
- Remove duplicates and clean inconsistent data


### 🛠️	Statistical Analysis and Hypothesis Testing

#### Description
Conduct statistical tests and probability analysis using numpy and scipy. Perform hypothesis testing to make data-driven conclusions and calculate confidence intervals to quantify uncertainty.

#### Requirements
Completed program should:

- Calculate probability distributions (normal, binomial, Poisson)
- Generate confidence intervals for population parameters
- Perform t-tests to compare means between groups
- Conduct chi-square tests for categorical data
- Calculate p-values and interpret significance levels
- Compute correlation and regression coefficients
- Perform ANOVA for multi-group comparisons
- Use simulation to estimate probabilities and validate results
- Document statistical assumptions and test appropriateness
