# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Master Python's fundamental data structures including lists, dictionaries, sets, and tuples. You will learn how to choose the right data structure for different problems, perform common operations efficiently, and understand the trade-offs between different collection types.

## 📝 Tasks

### 🛠️	Working with Lists and Tuples

#### Description
Explore mutable lists and immutable tuples to store and manipulate ordered collections of data. Learn list operations like sorting, filtering, and transforming data, as well as when to use tuples for immutable sequences.

#### Requirements
Completed program should:

- Create and manipulate lists using various methods (append, extend, insert, remove, pop)
- Perform list operations like slicing, indexing, and iteration
- Sort lists with custom sorting criteria
- Filter lists using list comprehensions and built-in functions
- Understand and use tuples for fixed collections of data
- Unpack tuple values and use them in function arguments


### 🛠️	Mastering Dictionaries

#### Description
Build proficiency with dictionaries to store and retrieve key-value pairs. Implement practical solutions using dictionary methods, handle nested dictionaries, and solve problems that require efficient lookups.

#### Requirements
Completed program should:

- Create dictionaries and access values using keys
- Use dictionary methods (keys(), values(), items(), get(), pop())
- Iterate through dictionaries efficiently
- Work with nested dictionaries
- Use dictionaries for counting and grouping data
- Implement dictionary comprehensions
- Handle missing keys gracefully with default values


### 🛠️	Sets and Advanced Operations

#### Description
Use sets for storing unique values and performing mathematical set operations. Learn to solve problems involving membership testing, removing duplicates, and combining collections using set operations like union, intersection, and difference.

#### Requirements
Completed program should:

- Create sets and add/remove elements
- Perform set operations (union, intersection, difference, symmetric difference)
- Check set membership efficiently
- Remove duplicates from collections using sets
- Understand set performance characteristics vs lists and dictionaries
- Use set comprehensions to build sets efficiently
- Combine multiple data structures effectively in problem solving
