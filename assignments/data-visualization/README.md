# 📘 Assignment: Python Data Visualization

## 🎯 Objective

Learn to create professional data visualizations using Python libraries like matplotlib and plotly. You will master different types of charts and graphs, customize visual elements for clarity and aesthetics, and effectively communicate data insights through compelling visualizations.

## 📝 Tasks

### 🛠️	Creating Charts with Matplotlib

#### Description
Build foundational skills using matplotlib to create various types of charts including line plots, bar charts, histograms, and scatter plots. Learn to customize colors, labels, legends, and formatting to make visualizations clear and informative.

#### Requirements
Completed program should:

- Create line plots showing trends over time
- Generate bar charts for comparing categorical data
- Build histograms to visualize distributions
- Create scatter plots to show relationships between variables
- Customize axes labels, titles, and legends
- Apply different colors, line styles, and markers
- Save plots to image files in multiple formats
- Create subplots to display multiple visualizations


### 🛠️	Advanced Visualizations with Plotly

#### Description
Explore interactive visualizations using plotly to create dynamic, web-ready charts. Build dashboards and interactive plots that allow users to explore data through hover information, zoom, and selection capabilities.

#### Requirements
Completed program should:

- Create interactive bar charts, line plots, and scatter plots
- Implement hover tooltips with detailed information
- Configure zoom and pan functionality
- Use dropdown menus and range sliders for interactivity
- Apply professional color schemes and styling
- Generate responsive plots that work on different screen sizes
- Export visualizations as HTML files
- Combine multiple plots in dashboard layouts


### 🛠️	Data Visualization Best Practices

#### Description
Apply design principles and best practices to create effective data visualizations. Build visualizations that accurately represent data, guide viewers to key insights, and follow accessibility and clarity standards.

#### Requirements
Completed program should:

- Choose appropriate chart types for different data distributions
- Use color effectively to highlight patterns and avoid confusion
- Implement proper scaling and axis ranges
- Add context with titles, annotations, and legends
- Create visualizations for different data types (numerical, categorical, temporal)
- Design for accessibility (colorblind-friendly palettes)
- Compare different visualization approaches for the same dataset
- Document visualization choices and design decisions
